PROJETO SIGEX
FAPES/TJES
Abril 2023 - Dezembro 2023

  1. SIGEX é o sistema que está sendo desenvolvido para a utilização dos cartórios do estado do Espírito Santo, Brasil. O sistema tem o objetivo de monitorar processos e serventias.
  2. ● Trabalhei como bolsista no ensino médio. O projeto contava com uma grande equipe de pessoas experientes e outros bolsistas do ensino médio.
  3. ● Utilizando de um ambiente no Docker para o seu desenvolvimento, o projeto conta com as tecnologias de Angular no front-end, e Spring para tratar do back-end, além de HTML, SASS e TypeScript.
  4. ● Bancos de Dados em SQL também foram utilizados, principalmente para o teste de APIs com Swagger.
  5. ● Desenvolvi testes de diversas funcionalidades com Cypress. Todas as atividades eram divididas entre equipes, e em forma de Sprint para o desenvolvimento de quaisquer atividades.

Desenvolvedor Arduíno

PROJETO MONITORAMENTO DE SINAIS VITAIS
FAPES
Abril 2022 - Dezembro 2022

  1. Este projeto tinha o objetivo de construir um dispositivo com Arduíno para o monitoramento de Sinais Vitais. Foi desenvolvido um bracelete que tinha a capacidade de medir batimentos cardíacos, oxigenação do sangue, ler o movimento, temperatura e localização.
  2. ● Trabalhei como bolsista no ensino médio. Juntamente com um instrutor e uma equipe.
  3. ● Tarefas eram divididas entre nós para a programação de cada um dos sensores utilizados.
  4. ● Foram utilizados microcontroladores ESP32 e diversos sensores de medição.
  5. ● Um aplicativo em JAVA foi desenvolvido para fácil monitoramento. O programa em comunicação com o microcontrolador mostrava todos os dados em tempo real e criava relatórios diários.
  6. ● Para a montagem final do dispositivo, foi modelado e impresso em 3D, um suporte que se prendia ao braço do usuário.
